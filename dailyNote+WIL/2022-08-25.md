## Java의 정석

### CH.1 자바를 시작하기 전에 
**자바의 특징**
	1. 운영체제에 독립적이다. 
	2. 객체 지향언어이다. 
	3. 비교적 배우기 쉽다. 
	4. 자동 메모리 관리 
	5. 네트워크 분산 처리를 지원한다. 
	6. 멀티 쓰레드를 지원한다. 
	7. 동적 로딩을 지원한다. 

**자바로 프로그램 작성하기**
Hello.java --javac.exe--> Hello.class --java.exe--> "hello world"
	1. 모든 코드는 클래스에 존재한다. 
	2. 클래스가 모여서 하나의 앱을 만든다. 
	3. 하나의 앱에는 main 메서드가 있어야한다. 
	4. 소스파일의 이름은 public class이름과 일치해야한다.

**변수, 상수, 리터럴**
